Welcome to SQL Server DevOps

The purpose of this project is to develop a set of PowerShell scripts that allow you to use XML files to define the configuration you want for MS SQL Server instances. This allows you to put your configuration of your SQL Servers under version control with all the associated befits. You can then use the scripts push that configuration to your servers.

Note: This project is just getting started. Contribution be they in the form of new scripts, patches, bug reports, suggestions for new features or just kind words of encouragement are welcomed. This project, like many is to scratch and itch I've had and maybe it can be of use to you too.

Getting Started

Requirements

The PowerShell Community Extensions (PSCX) module is required. The PSCX module needs to be found under the PSModulePath.aspx) environment variable.

Installation

  1. Download the latest version of the scripts using the buttons to the right.
  2. Unzip in your choice of location

Note: Not sure if there are issues with signing scripts and the like. If you have issues let me know or log a bug here.

Usage

After that you can run one of the following scripts:

  • SQLAgentDeployer.ps1 - Will configure a SQL Agent Instance
  • SQLRefreshEnviroment.ps1 - Used to refresh a non-production environment from backups (Not yet fully implemented)
  • SQLReplicationDeployer.ps1 - Will configure SQL Replication (Not yet fully implemented)
